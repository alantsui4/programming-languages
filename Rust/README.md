1. Installation
	```sh
	curl https://sh.rustup.rs -sSf | sh
	source $HOME/.cargo/env
	```
2. Setup toolchain
	```sh
	rustup default nightly
	```
3. Setup Clion
	```sh
	#Toolchain Location
	/home/alan/.cargo/bin
	```
4. Setup std lib
	```sh
	#Std lib location
	Download via rustup
	```

5. Create new projects
   ```sh
   cargo +nightly new helloWorld
   ```
